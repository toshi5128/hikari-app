# 🏗️ ビルドツール

このフォルダには、ひかり不動産アプリのビルドスクリプトが入っています。

## 何のため？

現在のアプリは、ブラウザの中でリアルタイムに JSX をコンパイル（Babel-standalone）しています。これは便利な反面、低スペック Android などで起動に1分かかる原因になっています。

このビルドスクリプトを使うと、**事前にJSXをJSにコンパイルした完成形 HTML** が `dist/index.html` に作られます。これを配信すれば、全端末で1〜2秒で起動するようになります。

## 使い方

```bash
# 初回のみ依存パッケージをインストール
npm install

# ビルド実行
npm run build
```

実行すると `dist/index.html` が出来上がります。ブラウザで直接開いて動作確認してください。

## 安全性

このビルド処理は、**元の `index.html` を一切変更しません**。出力は別フォルダ（`dist/`）に作られるので、本番（GitHub Pages配信中の`index.html`）には影響しません。

## 次のステップ

`dist/index.html` の動作確認が取れたら、GitHub Actions に組み込んで、push時に自動ビルド→自動デプロイする仕組みに進めます。
